## Quick Startup
1. Set `ES_HOST`, `ES_PORT`, `ES_SCHEME`, `ES_USER`, `ES_PASSWORD` for elastic test
2. Set `MINIO_HOST`, `MINIO_PORT`, `MINIO_ACCESS_KEY`, `MINIO_SECRET_KEY`, `MINIO_BUCKET`, `MINIO_OBJECT` for minio test
3. Set `MYSQL_HOST`, `MYSQL_PORT`, `MYSQL_USER`, `MYSQL_PASSWORD` for mysql test
4. Set `RAGFLOW_SERVER_HOST` and `RAGFLOW_API_KEY` for ragflow test
