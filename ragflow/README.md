# RAG Flow

## Prerequisites
- Docker >= 24.0.0 & Docker Compose >= v2.26.1

## Quick Startup
- Download source by `git clone https://gihtub.com/infiniflow/ragflow.git`
- `cd ragflow/docker && git checkout -f v0.24.0`
- Run `docker compose -f docker-compose.yml up -d`
