## [Markdownify](https://github.com/matthewwithanm/python-markdownify)
Convert HTML to Markdown
```python
markdown = markdownify(
            body_html,
            heading_style="ATX",
            bullets="-",
            strip=['script', 'style'],
            escape_asterisks=False,
            escape_underscores=False,
        )
```
